Como é uma estrutura básica de um HTML?

Voltar

O arquivo HTML é formado da seguinte estrutura: <html>, <head>, <body>.

  • A tag <html> indica que o arquivo se trata de uma estrutura HTML
  • Na tag <head> definimos a língua que está sendo feita o código, links de acesso à outras pastas, arquivos e url e também onde definimos o título da aba de nossa página na internet
  • Na tag <body> definimos a estrutura de nosso arquivo HTML

Entendendo a diferença entre listas ordenadas e não ordenadas

Voltar

Listas ordenadas são listas com uma ordem a ser seguida. Representadas pela tag <ol>, seguida pela tag <li> para a relação da lista.

Com o atributo type podemos ordenar:

  • type = "i" - ordena em algarismos romanos minúsculo.
  • type = "I" - ordena em algarismos romanos maiúsculo.
  • type = "a" - ordena com letras minúsculo.
  • type = "A" - ordena com letras romanos maiúsculo.
  • type = "1" - ordena com números.

Listas nâo-ordenadas são listas fora de ordem. Representadas pela tag <ul>, seguida pela tag <li> para a relação da lista.

Com o atributo type podemos apresentar a lista:

  • type = "disc" - com marcadores de bolinhas pretas.
  • type = "circle" - com marcadores com bolinhas sem preenchimento.
  • type = "square" - com marcadores de quadrados.
